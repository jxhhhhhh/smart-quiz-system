## 如何生成图片

### 方式一：VS Code 插件（推荐）
1. 安装 VS Code 插件 **PlantUML**
2. 安装 Java 运行环境（JRE 8+）
3. 打开 `.puml` 文件，按 `Alt+D` 预览，右键导出 PNG/SVG

### 方式二：在线渲染
1. 访问 [PlantUML Online Server](http://www.plantuml.com/plantuml/uml/)
2. 粘贴 `.puml` 文件内容
3. 下载生成的图片

### 方式三：命令行
```bash
# 安装 PlantUML
pip install plantuml

# 批量生成 PNG
java -jar plantuml.jar *.puml

# 或使用 Python
python -m plantuml *.puml
```

## 注意事项

- 所有 `.puml` 文件使用 UTF-8 编码
- 中文字体需要系统安装支持中文的字体（如微软雅黑）
- 建议输出格式为 PNG（300dpi）或 SVG（矢量图）
- 图 11（界面截图）需要手动截图，PlantUML 仅提供截图指导清单
