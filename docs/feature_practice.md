# 智能刷题系统 — 功能文档

## 1. 功能概述

智能刷题系统是一个面向高校学生的综合性学习辅助平台，集题目导入、智能解析、多模式练习、科学复习、数据统计于一体。系统支持**五种题型**：单选题、多选题、判断题、填空题、简答题。

## 2. 支持的题型

| 题型 | 标识 | 识别特征 | 答案格式 |
|------|------|---------|---------|
| 单选题 | `single` | 有 A/B/C/D 选项，答案为单字母 | `A` |
| 多选题 | `multi` | 有选项，答案含多个字母 | `ABD` |
| 判断题 | `judge` | 选项为"正确/错误"或答案含√/× | `A` 或 `对` |
| 填空题 | `fill` | 含 `___` 或无选项 | 任意文本如 `len` |
| 简答题 | `short` | 含"简答/论述/问答"等标签 | 较长文本回答 |

## 3. 支持的练习模式

### 3.1 随机模式 (random)
- **功能描述**：从题库中随机抽取指定数量的题目
- **适用场景**：日常练习、综合复习
- **支持按题型筛选**：可选择只练单选/多选/判断/填空/简答

### 3.2 错题重练模式 (wrong)
- **功能描述**：优先展示用户历史上答错次数最多的题目
- **适用场景**：针对性复习薄弱知识点
- **入口**：刷题练习页 或 错题回顾页的"开始错题练习"按钮
- **支持自定义数量**：错题回顾页可设置练习题目数量（1 到总错题数）

### 3.3 难度专项模式 (difficulty)
- **功能描述**：按难度等级筛选题目进行练习
- **难度等级**：1 = 简单，2 = 中等，3 = 困难

### 3.4 练习进度持久化
- **功能描述**：练习进度自动保存到数据库，中断后可恢复
- **适用场景**：浏览器意外关闭、页面刷新等中断情况
- **恢复方式**：再次进入练习页面时自动检测未完成进度，提供"继续上次练习"按钮

## 4. 导入题目

### 4.1 支持的导入方式

| 方式 | 说明 |
|------|------|
| 文本粘贴 | 直接粘贴学习通复制的文本 |
| PDF 文件 | 上传 `.pdf` 文件自动解析 |
| Word 文档 | 上传 `.docx` 文件自动解析 |
| 图片识别 | 上传 `.png/.jpg/.jpeg/.bmp` 图片，使用多模态LLM识别 |

### 4.2 解析模式

| 模式 | 说明 | 适用场景 |
|------|------|---------|
| 正则解析 | 基于正则表达式的快速解析 | 格式规范的标准文本 |
| AI 智能解析 | 调用大语言模型 API 智能识别 | 格式不规范、乱序、复杂文本 |
| 双层混合 | 正则优先 + AI 兜底 | 通用场景（推荐） |
| OCR 图片识别 | 多模态 LLM Vision API | 扫描件、截图、拍照 |

### 4.3 导入流程

1. **选择学科分类**（必填）：选择已有学科或自定义输入
2. 选择解析模式（正则 / AI / 双层）
3. 如果选择 AI 模式，配置 API Key（默认 DeepSeek）
4. 粘贴文本 / 上传文件 / 上传图片
5. 点击"🔍 解析预览"查看解析结果
6. 在预览区可**行内编辑**每道题（题型、内容、选项、答案、难度）
7. 支持**批量操作**：批量删除、批量设置难度、批量设置学科
8. 确认无误后点击"✅ 确认导入"

### 4.4 文本格式示例

```
1. Python是一种解释型语言。
A. 正确
B. 错误
答案：A

2.下列属于Python内置数据类型的是（多选）
A. list
B. dict
C. array
D. tuple
答案：ABD

3、_____ 函数用于获取列表长度。
答案：len

4.（简答题）请简述Python中列表和元组的区别。
答案：列表可变，元组不可变。
```

### 4.5 智能解析特性（正则模式）

- **全角→半角自动转换**：`ＡＢＣ` → `ABC`
- **题型自动识别**：无需手动标注，系统根据选项和答案自动判断
- **答案标准化**：多选排序去重（`BDA` → `ABD`）、判断统一符号、填空忽略大小写
- **简答题识别**：自动识别"简答/论述/问答/简述/解释/分析"等标签
- **正则预编译**：20+ 个高频正则表达式在模块加载时预编译，提升解析性能

### 4.6 AI 智能解析特性

- **格式无关**：无论文本格式是否规范，AI 均可理解并提取题目
- **结构化输出**：AI 返回标准 JSON 格式，包含题型、内容、选项、答案
- **支持乱序**：可处理选项乱序、题号缺失等复杂情况
- **自动清洗**：忽略教师批阅标记、得分信息等无关内容
- **多模型支持**：默认 DeepSeek，可切换为 OpenAI、通义千问等

### 4.7 OCR 图片识别特性

- **多模态 LLM**：使用支持 Vision 的大语言模型直接识别图片内容
- **多格式支持**：PNG、JPG、JPEG、BMP
- **自动检测**：根据图片文件头自动检测图片类型
- **中文优化**：DeepSeek/GPT-4o 等模型对中文题目识别效果优秀
- **直接入库**：识别结果经标准化后直接进入导入流水线

### 4.8 答案判断逻辑

填空题/简答题的答案判断已优化：
- 忽略括号差异：`len()` 和 `len` 视为相同
- 支持包含匹配：用户答案包含正确答案，或反之
- **短字符串保护**：当正确答案和用户答案都 ≤3 字符时，跳过包含匹配，避免 `id` 匹配 `hidden` 等误判
- 多答案模式：逗号/分号分隔后支持子集匹配
- Levenshtein 模糊匹配：容忍拼写错误（如 `pirnt` → `print`）

## 5. Web 界面功能

### 5.1 用户认证
- **注册**：用户名（≥2字符）+ 密码（≥4字符）+ 显示名称
- **登录**：SHA-256 + 随机盐密码哈希验证
- **会话管理**：登录后侧边栏显示用户名，支持退出登录
- **数据隔离**：答题记录、收藏、导入历史按用户隔离

### 5.2 仪表盘
- 题库总量、答题总次数、总体正确率、学科分类数
- 题型分布统计
- 快速开始入口

### 5.3 导入题目
- 文件上传（PDF/DOCX/图片）+ 文本粘贴
- **学科必填**：导入前必须选择学科分类
- **多模式解析**：正则 / AI / 双层 / OCR
- **AI 配置**：支持 DeepSeek / OpenAI / 自定义 API
- **解析预览**：导入前可查看解析结果，支持行内编辑
- **批量操作**：批量删除、批量设置难度、批量设置学科
- 导入历史记录

### 5.4 刷题练习
- 三种模式：随机/错题/难度
- **题型筛选**：全部/单选/多选/判断/填空/简答
- 题目数量设置（1-100）
- **本题计时** + **总用时统计**
- 多选题使用 checkbox，填空题/简答题使用文本框
- 提交后禁用选项，防止误操作
- **中途退出**：练习过程中可随时退出
- **进度持久化**：练习进度自动保存，支持中断恢复
- 练习完成显示评价 + "再来一组" / "返回设置"按钮

### 5.5 模拟考试
- 考试设置：题数、时长、题型、学科
- **倒计时显示**：实时显示剩余时间
- **低时间警告**：剩余 <5 分钟时发出警告
- **超时自动交卷**：JavaScript 定时器确保超时自动提交
- 考试结果：成绩、正确率、用时

### 5.6 间隔重复复习
- **艾宾浩斯遗忘曲线**：第 1、2、4、7、15、30 天复习提醒
- 题目分类：今日到期、即将到期、已掌握、无错题
- 按学科筛选
- 一键开始复习

### 5.7 学习统计
- 近 30 天学习数据卡片（答题总次数、正确数、正确率）
- 近 7 天每日答题数量柱状图
- 近 7 天每日正确率趋势折线图
- **各题型正确率柱状图**
- 最近答题记录（含题型标签）
- **智能学习建议**：基于题型弱项、趋势分析、错题数量

### 5.8 错题回顾
- 错题列表（按错误次数排序）
- 显示题型标签、正确答案、错误次数
- **自定义练习数量**：可设置错题练习的题目数量
- **一键开始错题练习**按钮

### 5.9 题目管理
- 按学科/题型/关键词筛选，支持收藏筛选
- **分页导航**：上一页/下一页按钮 + 页码输入
- 查看题目详情（题干、选项、答案）
- **编辑题目**：修改内容、选项、答案、难度、学科
- **删除题目**：单题删除或批量删除当前页（同时清除相关答题记录和收藏）
- **收藏题目**：切换收藏状态
- **导出题库**：TXT 文件导出 + 全量 JSON 导出

### 5.10 答题历史
- 按时间范围筛选（7天/30天/90天/全部）
- 按答题结果筛选（全部/正确/错误）
- 显示总记录数、正确数、正确率统计（基于全量筛选结果）
- **分页导航**：上一页/下一页按钮 + 页码输入
- **导出答题记录**为 CSV 文件（UTF-8 BOM 编码，兼容 Excel）
- 每条记录显示题型、题目、用户答案、正确答案、用时

### 5.11 系统功能
- **深色/浅色主题切换**：侧边栏一键切换，全局 CSS 适配
- **移动端响应式布局**：768px/480px 断点媒体查询
- **统计图表**：使用 Plotly 生成交互式图表，自动适配主题

## 6. 数据库结构

系统使用 SQLite3 数据库，共 6 张表：

### 6.1 questions 表（题目）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| subject | TEXT | 学科分类 |
| question_content | TEXT | 题目内容 |
| options | TEXT | 选项（JSON） |
| correct_answer | TEXT | 正确答案 |
| difficulty | INTEGER | 难度（1-3） |
| question_type | TEXT | 题型（single/multi/judge/fill/short） |
| source | TEXT | 来源 |
| tags | TEXT | 标签 |
| ai_enhanced | INTEGER | 是否已AI增强 |
| chapter | TEXT | 章节 |
| fingerprint | TEXT | 语义指纹（MD5） |
| created_at | TEXT | 创建时间 |

### 6.2 practice_records 表（答题记录）
| 字段 | 类型 | 说明 |
|------|------|------|
| record_id | INTEGER | 主键 |
| question_id | INTEGER | 题目ID（外键） |
| user_id | INTEGER | 用户ID |
| user_answer | TEXT | 用户答案 |
| is_correct | INTEGER | 是否正确 |
| time_spent | REAL | 答题用时（秒） |
| practice_date | TEXT | 答题时间 |

### 6.3 favorites 表（收藏）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| question_id | INTEGER | 题目ID（外键，唯一） |
| user_id | INTEGER | 用户ID |
| created_at | TEXT | 收藏时间 |

### 6.4 import_history 表（导入历史）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| source | TEXT | 来源文件名 |
| count | INTEGER | 导入数量 |
| subject | TEXT | 学科 |
| parse_mode | TEXT | 解析模式 |
| user_id | INTEGER | 用户ID |
| imported_at | TEXT | 导入时间 |

### 6.5 users 表（用户）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| username | TEXT | 用户名（唯一） |
| password_hash | TEXT | 密码哈希 |
| display_name | TEXT | 显示名称 |
| role | TEXT | 角色 |
| created_at | TEXT | 注册时间 |

### 6.6 practice_sessions 表（练习会话）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| session_key | TEXT | 会话标识 |
| questions_json | TEXT | 题目快照 |
| current_index | INTEGER | 当前进度 |
| score | INTEGER | 当前得分 |
| mode | TEXT | 练习模式 |
| settings_json | TEXT | 设置 |
| user_id | INTEGER | 用户ID |
| created_at | TEXT | 创建时间 |
| updated_at | TEXT | 更新时间 |

## 7. 文件结构

```
├── config.py              # 统一配置文件
├── database.py            # 数据库管理（单例+线程安全+6张表）
├── parser.py              # 正则解析引擎（预编译+答案判断+模糊匹配）
├── ai_parser.py           # AI解析引擎（文本+图片OCR）
├── ai_enhancement.py      # AI增强模块
├── question_importer.py   # 题目导入器（双层解析+去重+质量检查）
├── practice_session.py    # 刷题逻辑
├── styles.py              # 主题样式（深色/浅色+响应式）
├── logger.py              # 日志模块
├── main.py                # CLI入口
├── web_interface.py       # Web入口（路由+认证）
├── page_modules/          # 页面模块（9个页面）
├── tests/test_system.py   # 44个单元测试
├── requirements.txt       # 依赖列表
└── data/question_bank.db  # SQLite数据库
```
